## Getting Started

**Set up environment**

```bash
vercel login
cd apps/web
vercel link
vercel env pull .env.development.local
```

**Start the development server**

```bash
pnpm dev
```

Open [http://localhost:3004](http://localhost:3004) with your browser to view the app.
